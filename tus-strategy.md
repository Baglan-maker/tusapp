# «Түс» — Стратегия разработки, архитектура и бизнес-план

Соло-основатель, side-project, стек: React Native + TypeScript + FastAPI + Python + PostgreSQL. Всё вайбкодится через Claude Code. Целевой рынок: русскоязычный СНГ + казахский как незанятая ниша, английский — фаза 3.

## 1. КастДев: как проверить до того, как писать код

### Правило №1 — Mom Test
Не спрашивай «купил бы ты приложение-сонник?» — все скажут «да» и наврут. Спрашивай про **прошлое поведение**:

1. «Когда тебе последний раз снился странный сон — что ты сделал(а) после пробуждения?»
2. «Открывал(а) ли ты когда-нибудь сонник? Какой, где, зачем?»
3. «Гуглил(а) "к чему снится…"? Что именно, покажи историю»
4. «Записываешь ли сны? Куда? Почему бросил(а)?»
5. «Платишь ли за какие-нибудь подписки в приложениях? Какие?»
6. «Рассказываешь ли сны друзьям / в сторис?»

Если человек ни разу не гуглил сон и не открывал сонник — он не твоя аудитория, дальше не трать время.

### План кастдева (2 недели, параллельно с MVP)
- **15–20 интервью по 20 минут.** Сегменты: (а) девушки 18–30 из астро/эзотерика-аудитории RU/KZ — ядро платящих в вертикали; (б) мусульманский сегмент (интерес к Ибн Сирину); (в) люди, ведущие дневники/журналы.
- **Где искать респондентов:** комментарии под сонник-блогерами в Instagram/TikTok, Telegram-каналы по эзотерике и психологии, твои ученики и их родители, студенческие чаты AITU. Оффер: «делаю приложение про сны, 20 минут разговора — растолкую твой сон бесплатно».
- **Смок-тест ценой:** простой лендинг (Tilda/Framer, 1 вечер) с описанием, скринами из Figma и кнопкой «Получить ранний доступ — 1490 тг/мес после запуска». Меришь: сколько из зашедших оставили почту/телефон. 3 TikTok-видео для трафика. Конверсия в waitlist >8% — сильный сигнал.
- **Telegram-бот как concierge-MVP (главный лайфхак для СНГ).** За 1–2 дня собери бота: пользователь шлёт голосовое → Whisper → LLM → толкование текстом. Ноль сторов, ноль модерации, мгновенная дистрибуция в чатах. Это проверит retention-гипотезу (возвращаются ли на 3-й, 7-й день) ещё ДО того, как ты напишешь мобильное приложение. Если 50 человек попробовали и D7 < 10% — чини продукт, а не строй приложение.

### Критерии «go» после кастдева
- ≥60% интервьюируемых гуглили сны за последние 3 месяца
- Waitlist-конверсия лендинга ≥8%
- D7 Telegram-бота ≥15%
- Хотя бы 3 человека спросили «когда можно заплатить?»

## 2. Продукт: фичи, которые выделяют

### Ядро (MVP, недели 1–4)
| Фича | Зачем |
|---|---|
| **Sleepy Mode: запись в полусне** | Открыл приложение → сразу огромная кнопка записи на тёмном экране, никаких меню. Виджет на локскрине iOS / quick tile Android. 70% деталей сна забываются за 10 минут — скорость捕жения = твой продукт |
| **Голос → транскрипт → толкование за 60 сек** | Whisper (рус — отлично, каз — приемлемо) → структурированное AI-толкование |
| **Мульти-линза толкования** | Один сон — выбор призмы: 🧠 Психологическая (Юнг/современная), 📖 Классика (Миллер/Ванга), ☪️ По Ибн Сирину (режим «справочник традиции», не «AI выносит вердикт» — снимает фетва-риск), 🔬 Научная (нейробиология сна). Ни у одного конкурента нет такого выбора |
| **Паттерн-движок** | «Вода снится тебе 3-й раз за месяц — и каждый раз в дни, когда ты отмечал тревожность». Embeddings + извлечение символов из каждого сна. Накопительная ценность = anti-novelty |
| **Shareable-карточка** | Красивая карточка: символ сна + одна строка толкования + брендинг. Кнопка шеринга сразу после генерации + мягкий промпт «отметь @tus.app» (петля Co-Star) |
| **Стрики + утренний пуш** | Пуш привязан к времени пробуждения пользователя (спрашиваем в онбординге). Duolingo-механика против novelty-эффекта |

### Дифференциаторы (фаза 2, недели 5–10)
| Фича | Зачем |
|---|---|
| **Ай-визуализация сна** (картинка по описанию) | Самый виральный формат для TikTok: «я рассказал ИИ свой сон, и вот что он нарисовал». Premium или 1/неделю бесплатно |
| **Месячный Dream Report («Spotify Wrapped снов»)** | Топ-символы, карта эмоций, паттерны, инсайт месяца. Shareable, premium |
| **AI-доуточнение (чат по сну)** | «А что значит, что дом был бабушкин?» — 3 вопроса бесплатно, безлимит в premium |
| **Эмоции + теги при записи** | 2 тапа после записи: эмоция сна, эмоция дня. Топливо для паттерн-движка и графиков |
| **Казахский как first-class** | Полный UI + толкования + ASO на казахском. Конкуренты на казахском — статические справочники без AI и голоса. Это твой пустой плацдарм |
| **Лунный календарь / «вещие дни»** | Дешёвая фича, любима аудиторией сонников (день недели/лунный день влияет на «значимость» сна в народной традиции) |
| **Приватность как фича** | Сны — интимный контент. On-device черновики, экспорт, «удалить всё» одной кнопкой. Прямо коммуницировать |

### Фаза 3 (после PMF, месяцы 4+)
- HealthKit/Google Fit: корреляция снов с качеством сна
- Анонимная лента «сны мира» (модерация LLM)
- Английская локаль + мировой запуск
- Совместимость снов пары (виральная механика в духе Co-Star)

### Free vs Premium
**Free:** 1 толкование/день (1 линза), дневник безлимитно, стрики, 1 карточка/день.
**Premium (1490 тг/мес или 9 990 тг/год, триал 7 дней):** безлимит толкований, все линзы, паттерн-отчёты и Dream Report, AI-чат, визуализация снов, полная история и поиск.
Ранним пользователям — lifetime за 14 990 тг (первые 200): кэш на старте + евангелисты.

## 3. Архитектура

### Принцип: минимум движущихся частей для соло + вайбкодинга

```
┌─────────────────────────────────────────────────┐
│  МОБИЛКА: Expo (React Native + TypeScript)      │
│  expo-av (запись) · RevenueCat SDK · Expo Push  │
│  Sentry · PostHog                               │
└──────────────────┬──────────────────────────────┘
                   │ HTTPS / JWT (Supabase Auth)
┌──────────────────▼──────────────────────────────┐
│  BACKEND: FastAPI (Python) на Railway/Fly.io    │
│  /dreams  /interpret  /patterns  /cards  /quota │
│  Celery-beat или APScheduler: ночные джобы      │
│  (паттерн-анализ, месячные отчёты, пуши)        │
└───────┬──────────────────┬──────────────────────┘
        │                  │
┌───────▼────────┐  ┌──────▼───────────────────────┐
│ Supabase        │  │ AI-провайдеры                │
│ · Postgres      │  │ · STT: Groq whisper-large-v3 │
│   + pgvector    │  │   (фолбэк OpenAI Whisper)    │
│ · Auth (Apple/  │  │ · LLM: Claude Haiku (free    │
│   Google/email) │  │   tier) / Sonnet (premium)   │
│ · Storage       │  │ · Embeddings:                │
│   (аудио, PNG)  │  │   text-embedding-3-small     │
└─────────────────┘  │ · Image gen (фаза 2)         │
                     └──────────────────────────────┘
Платежи: RevenueCat → App Store / Google Play IAP
```

**Почему так:**
- **Supabase** закрывает auth, базу, storage и RLS — минус ~2 недели соло-работы. Postgres внутри — твой родной стек, pgvector для паттернов из коробки.
- **FastAPI отдельным сервисом** — весь AI-пайплайн (STT → извлечение символов → толкование → карточка) в одном месте, легко логировать/кэшировать/менять модели. Твой родной фреймворк.
- **Groq Whisper** — ~$0.002/мин, latency в секунды. Для казахского качество проверить на бете; фолбэк — OpenAI.
- **RevenueCat** — обязательно: paywall-эксперименты, триалы, аналитика подписок без своего биллинга.
- **Пайплайн одного сна:** аудио → STT → LLM-джоб №1 (structured output: символы, эмоции, сущности → JSON) → LLM-джоб №2 (толкование в выбранной линзе, с контекстом последних снов и паттернов юзера — лечит «однообразие», главную жалобу на конкурентов) → embedding в pgvector → ответ.
- **Себестоимость толкования:** STT ~$0.002 + LLM ~$0.003 + embedding ~$0.0001 ≈ **$0.005–0.01**. Free-юзер с 30 толкованиями/мес ≈ $0.2–0.3 — приемлемо.

Ключевые решения в схеме:
- **Символы нормализованы** (SYMBOLS + DREAM_SYMBOLS) — паттерн-движок ищет повторы SQL-запросом, а не по тексту.
- **PATTERNS.last_notified** — чтобы не спамить одним инсайтом («вода 3-й раз») каждый день.
- **USAGE_QUOTA по дням** — квота free-тарифа считается на бэке, не в клиенте.
- **SHARE_CARDS.shared_at** — метрика K-фактора: сколько карточек реально расшарено.
- **Soft delete снов** — приватность + «удалить всё» без каскадных сюрпризов.

## 4. Бизнес-план

### Расходы (месячно, side-project режим)
| Статья | Сумма |
|---|---|
| Railway/Fly + Supabase Pro | $25–45 |
| AI API (при 2K MAU, ~60% активных) | $50–150 |
| Apple Developer | $99/год (~$8/мес) |
| Google Play | $25 одноразово |
| RevenueCat, PostHog, Sentry | $0 (free tiers до серьёзных объёмов) |
| **Итого** | **~$85–200/мес** |

Точка безубыточности: **~40–90 платящих подписчиков.**

### Юнит-экономика
- Цена: 1490 тг/мес ≈ $3. Нетто после комиссии стора (15% для small business program): ~$2.5
- Годовая 9 990 тг ≈ $20 → нетто ~$17 (пуш в годовую — стандарт вертикали, режет churn)
- Себестоимость premium-юзера (безлимит, реально ~40–60 толкований/мес): $0.3–0.6 → **маржа ~75–85%**

### 12-месячные сценарии (из ресерча: конверсия 2–4%, D30 ~3–5%)
| | Пессимист | База | Оптимист (виральный хит) |
|---|---|---|---|
| Загрузки за год | 5K | 20K | 100K |
| Платящих к мес. 12 | 40 | 250 | 1 500 |
| MRR к мес. 12 | ~$100 | ~$600 | ~$3 500 |
| Вердикт | Закрыть или пивот | Хороший side-income, держать | Уходить в фултайм |

### Пороги решений (из ресерча, зашиты заранее)
- **Неделя 6 после запуска:** D7 < 10% → стоп-маркетинг, чинить ритуал (пуши, стрики, онбординг)
- **Месяц 3:** конверсия free→paid < 2% на активном ядре → тест hard paywall с триалом (медиана конверсии hard paywall 10.7% vs 2.1% freemium по RevenueCat)
- **Месяц 4:** K-фактор карточек ≈ 0 и органика мертва → менять контент-стратегию или закрывать
- **Месяц 6:** MRR < $150 при честно отработанной дистрибуции → проект в maintenance, силы на другое

### Дистрибуция без бюджета
1. **TikTok/Reels — главный канал.** Три формата: (а) «рассказал ИИ свой сон — вот толкование» (скринкаст), (б) визуализация сна картинкой (самый виральный), (в) разборы популярных снов («к чему снятся зубы — отвечает ИИ»). Русский + казахский контент раздельно.
2. **ASO на казахском** — «түс жору» и родственные запросы почти без конкуренции среди AI-приложений.
3. **Виральная петля в продукте** — карточка → шеринг → водяной знак → новые юзеры. Merишь через SHARE_CARDS.shared_at.
4. **Микроблогеры эзотерика/психология 5–50K подписчиков** — бартер: пожизненный premium за обзор.
5. **Сезонность:** пик интереса к снам — Рамадан (мусульманский сегмент) и январь–февраль. Подгадай крупные пуши контента.

### Риски и митигации
| Риск | Митигация |
|---|---|
| Novelty: бросают за 3–7 дней | Стрики + утренний пуш по wake_time + паттерн-инсайты как «сериал про себя» |
| Однообразие толкований (жалоба №1 на конкурентов) | Контекст юзера в промпте: последние сны, паттерны, эмоции. Ротация структуры ответа |
| Фетва-риск исламского режима | Режим = «справочник традиции Ибн Сирина», дисклеймер, AI не выносит религиозных суждений |
| Kovallux/DreamAI уже сделали голос+AI на русском | Твоё: мульти-линза, паттерн-движок, казахский, Sleepy Mode UX. Скорость: их продукты сырые, окно ~год |
| Whisper слаб на казахском | Тест на бете; фолбэк — ввод текстом + «продиктуй на русском, ответ на казахском» |
| Стоимость AI при росте free-юзеров | Квоты в USAGE_QUOTA, кэш популярных символов, дешёвые модели для free |
